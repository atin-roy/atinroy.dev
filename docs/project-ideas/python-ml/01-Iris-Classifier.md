# Iris Flower Classifier

**Tier:** 1-Beginner (ML)  
**Stack:** Python, scikit-learn, Pandas, Jupyter Notebook

A classic machine learning project using the Iris dataset to learn classification fundamentals.

## Learning Goals

- NumPy and Pandas data manipulation
- Data exploration and visualization
- Train/test splitting
- Classification algorithms (Decision Tree, Logistic Regression, KNN)
- Model evaluation metrics (accuracy, precision, recall, F1)
- Cross-validation
- Feature scaling and normalization

## Project Tasks

1. [ ] Load the Iris dataset
2. [ ] Explore the data (shape, types, statistics)
3. [ ] Visualize features using matplotlib/seaborn
4. [ ] Check for missing values and handle them
5. [ ] Split data into training and testing sets (80/20)
6. [ ] Train multiple classifiers:
   - Decision Tree
   - Logistic Regression
   - K-Nearest Neighbors
7. [ ] Evaluate models using accuracy, precision, recall, F1-score
8. [ ] Create confusion matrix for each model
9. [ ] Implement cross-validation
10. [ ] Save the best model using pickle

## Bonus Tasks

- [ ] Hyperparameter tuning with GridSearchCV
- [ ] Visualize decision boundaries
- [ ] Create a web app using Flask/Streamlit to predict iris type
- [ ] Compare training vs validation performance
- [ ] Implement feature selection
- [ ] ROC/AUC curve analysis
- [ ] Use ensemble methods (Random Forest, Gradient Boosting)
